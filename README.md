# 🎮 Jogo do Número Secreto

Um jogo simples e divertido onde o usuário deve adivinhar um número secreto entre **1 e 50**.  
O programa fornece dicas se o número escolhido é **maior** ou **menor** que o chute do jogador, até que ele acerte.  
Ao final, aparece a mensagem: **"Parabéns, você acertou!"** 🎉

---

## 🚀 Tecnologias utilizadas
- **HTML5** → estrutura da página  
- **CSS3** → estilização e design futurista  
- **JavaScript** → lógica do jogo e interatividade  

---

## 🕹️ Como jogar
1. Abra o jogo no navegador.  
2. Escolha um número entre **1 e 50**.  
3. Clique em **Adivinhar**.  
4. O jogo dará dicas se o número secreto é maior ou menor.  
5. Continue tentando até acertar!
